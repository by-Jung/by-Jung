Hi! I’m an Android / AOSP / Embedded developer who enjoys building reliable system-level software.  
I work across Android applications, framework layers, Linux-based environments, and network-connected embedded devices. 🚀

#### 📱 On the Android side,
I build Android apps with Java and work on system apps, services, and platform-integrated features.  
I’m especially interested in architecture, maintainable code, and stable interactions between UI, services, and device components.

#### ⚙️ On the platform & embedded side,
I work with AOSP, vendor customization, device features, and low-level integration.  
I’m comfortable reading and modifying framework / native / vendor code, and troubleshooting cross-layer issues from app to kernel-adjacent areas.

#### 🌐 On the networking side,
I work with Linux networking concepts such as routing, NAT, ROUTING, tethering, interface control, and packet flow analysis.  
I also handle device communication scenarios involving satellite modules and embedded connectivity.

#### 🛰️ On the domain side,
I’m interested in Android, hardware state handling, Network and real-world device behavior.  
I enjoy solving problems where software must work closely with hardware and unstable network conditions.

#### 💚 Off the clock,
I like organizing technical knowledge, writing things down clearly, and refining code until it becomes easier to read and maintain. ✨
